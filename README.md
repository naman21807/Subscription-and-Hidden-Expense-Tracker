**Subscription and Hidden Expense Tracker**
->This project contains OOPs implementation and Tkinter GUI
->This application helps user track their monthly subscriptions and calculate their total expenses, it also highlights hidden yearly costs that users ignore and point out the unused subscriptions.
**Objectives**
->To apply OOPs concept
->To design a GUI using tkinter library
->To help user manage and analyze subscription expenses
->To identify hidden charges and unused subscriptions
**Features**
->Add subscription(Name, Monthly cost, renewal date)
->View all subscriptions
->Calculate total monthly expense
->Calculate total yearly expense
->Identify upcoming renwal dates
->Identify unused subscriptions
->Track last used date for each subscription
